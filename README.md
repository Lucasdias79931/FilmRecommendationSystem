# Movie Recommendation System

## Overview

This project aims to implement a **movie recommendation system** using classic data structures such as linked lists, stacks, queues, and binary trees.

The goal is to apply these structures in a real-world scenario by building a system capable of recommending movies to users based on their preferences and interaction history.

## Objectives

* Practice and apply fundamental data structures
* Build a recommendation system from scratch
* Explore performance and organization using different structures
* Integrate a high-performance backend with a modern frontend

## Technologies

### Backend

* **C++**
* **gRPC**
* **oat++**

### Frontend

* **React**
## Data Structures Used

* **Linked List** → Stores users and movies  
* **Stack** → Manages user watch history  
* **Queue** → Handles recommendation requests  
* **B+ Tree** → Organizes and searches movies efficiently  
* **Decision Tree** → Builds filters to recommend movies to users  
* **Hash Table** → Implements a basic cache  
## System Architecture

* The frontend (React) communicates with the backend using **gRPC**
* The backend handles all business logic and data processing using C++ and data structures

## Features (Planned)

* User registration and management
* Movie catalog management
* Personalized movie recommendations
* Watch history tracking
